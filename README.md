# Spicy Braised Tofu — HTML Recipe Page

A recipe webpage built with HTML as part of the **WEB 101 Assignment** for the AltSchool Africa School of Engineering, Semester 1.

---

## 📄 About the Project

This project is a single-page HTML recipe website for **Spicy Braised Tofu**, adapted from [Chef Lola's Kitchen](https://cheflolaskitchen.com/spicy-braised-tofu/). It demonstrates core HTML concepts including document structure, semantic elements, text formatting, lists, images, tables, and links.
